# Editorial System

This area contains the end-to-end editorial lifecycle.

## Lifecycle

```text
Problem Discovery
  ↓
Research
  ↓
Decision Criteria
  ↓
Editorial Brief
  ↓
Draft
  ↓
Evidence and DNA Review
  ↓
Publish
  ↓
Measure
  ↓
Update or Retire
```
